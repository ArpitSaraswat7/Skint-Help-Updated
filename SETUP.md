## ⚠️ IMPORTANT: PowerShell Execution Policy Issue

Your system has a PowerShell execution policy that prevents running npm commands.

### 🔧 **SOLUTION - Choose ONE:**

#### Option 1: Run in Command Prompt (Easiest)
```cmd
# Open Command Prompt (cmd.exe) instead of PowerShell
cd "d:\Project's\skint help\Skint help new version"
npm run dev
```

#### Option 2: Temporarily Allow PowerShell Scripts
```powershell
# Run PowerShell as Administrator, then:
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

# Then run your project:
cd "d:\Project's\skint help\Skint help new version"
npm run dev
```

#### Option 3: Use VS Code Terminal
1. Open VS Code
2. Open the project folder
3. Use the integrated terminal (it often bypasses this issue)
4. Run `npm run dev`

## 🆘 **TROUBLESHOOTING**

### Issue: npm commands don't work
**Solution**: Use Command Prompt instead of PowerShell, or change execution policy

### Issue: Port 5173 already in use
**Solution**: Vite will automatically use next available port (5174, 5175, etc.)

### Issue: Styles not loading
**Solution**: Make sure Tailwind is processing correctly - check terminal for errors

### Issue: Animations laggy
**Solution**: Close other browser tabs, animations are GPU-accelerated
